# Rich Life Planner

A couples' web app for designing the second half of life together. Two partners
independently answer reflective exercises about the transitions ahead and the
life they want to build, then trigger a reveal to see each other's answers side
by side — with tailored observations generated by Claude.

Grounded in two bodies of work:

- **Module 1 — Transitions** (William Bridges' *Transitions*): Endings, the
  Neutral Zone, what you're not ready to let go of, what is beginning, the
  transition itself.
- **Module 2 — Vision** (Ramit Sethi's *I Will Teach You to Be Rich*): the Rich
  Life challenge, the future state, life chapters, Money Dials, the windfall.

## Run it now (demo mode, no setup)

```bash
python3 -m http.server 4321
# open http://localhost:4321 → "Continue in demo mode"
```

Demo mode runs entirely in the browser on `localStorage`. Use **Add a simulated
partner** and the **Acting as** switcher to try the full reveal flow solo.

## Go live

Fill in `js/config.js` and follow [SETUP.md](SETUP.md) for the four manual steps:
Supabase database (SQL + RLS), the Claude Edge Function, Google OAuth, and
GitHub Pages deployment.

## Tech

Vanilla HTML/CSS/JS + [Alpine.js](https://alpinejs.dev) · Supabase (Postgres,
Google OAuth, Edge Functions) · Claude `claude-haiku-4-5` via Edge Function ·
static, no build step. Answers save locally first and sync to Supabase when
configured; the app works fully without the Claude API.

## Layout

```
index.html                       app shell + all views (Alpine templates)
css/styles.css                   design system
js/config.js                     public runtime config (blank = demo mode)
js/content.js                    all module/exercise content
js/storage.js                    local-first persistence
js/app.js                        Alpine root component (all app logic)
js/backend.js                    Supabase auth/sync/Claude interface
supabase/functions/claude/       Edge Function (Claude proxy)
SETUP.md                         manual setup guide
transition-form-plan.md          original product plan
```
